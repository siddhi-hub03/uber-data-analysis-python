# Uber Data Analysis 🚗

## Project Overview
This project analyzes Uber trip data to identify ride patterns, peak hours, trip purposes, and high-demand locations using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified peak ride hours
- Analyzed trip distribution
- Visualized top pickup and drop locations
- Performed exploratory data analysis (EDA)

## Visualizations
- Heatmaps
- Bar charts
- Histograms
- Pie charts
- Trend analysis

## Project Conclusion
The analysis helps understand customer travel behavior and demonstrates practical data analytics skills.
